PROMPT 43: Show me the SQL and explain exactly which filters you used to calculate last 30 days revenue.

──────────────────────────────────────────────────────────────────────
To calculate the last 30 days revenue, the SQL query used is as follows:

```sql
SELECT SUM(totalAmount) AS totalRevenue
FROM "Order"
WHERE orderDate >= date('now', '-30 days') AND status = 'Delivered';
```

### Filters Used:

- **Date Filter**: 
  - `orderDate >= date('now', '-30 days')`: This filter selects orders that were placed in the last 30 days from the current date.

- **Status Filter**: 
  - `status = 'Delivered'`: This filter ensures that only orders that have been delivered are included in the revenue calculation. Cancelled or pending orders are excluded to provide an accurate revenue figure.

These filters together ensure that the revenue calculation reflects only completed transactions within the specified time frame.
